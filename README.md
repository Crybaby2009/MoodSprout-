# 🌱 MoodSprout

MoodSprout is a Slack bot that helps teams track wellbeing through a virtual plant.

Users can check in with moods such as:

* happy 😊
* excited 🎉
* stressed 😔
* tired 😴

Positive moods help the plant grow, while negative moods cause it to wilt.

## Features

* Mood check-ins
* Growing virtual plant
* Team wellbeing tracking
* Plant health system
* Slack integration

## Commands

* happy
* excited
* stressed
* tired
* garden
* help

## Tech Stack

* Python
* Slack Bolt
* Slack API

## Inspiration

MoodSprout makes team wellbeing fun and visible through a collaborative virtual plant.
